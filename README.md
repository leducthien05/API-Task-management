# 📋 Task Management RESTful API

## 📌 Description
Đây là RESTful API quản lý công việc được xây dựng bằng Node.js và Express. Hệ thống cho phép người dùng tạo, cập nhật, xóa và theo dõi trạng thái các công việc (tasks).

API được thiết kế theo chuẩn REST, dễ dàng tích hợp với frontend (web/mobile).

---

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB / Mongoose
- JWT Authentication
- RESTful API

---

## ✨ Features

- Tạo công việc mới
- Lấy danh sách công việc
- Cập nhật công việc
- Xóa công việc
- Đánh dấu trạng thái (pending / completed)
- Tìm kiếm và lọc công việc
- Xác thực người dùng (JWT)
